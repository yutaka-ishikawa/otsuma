tcp_perf1.cとtcp_perf2.cのプログラムをコンパイルして実行しましょう。
今回、Makefileというファイルもあります。
1. makeコマンドを実行すると、tcp_perf1.c, tcp_perf2.cの両方をコンパイルします。makeコマンドのインストールはAIに聞きましょう。
2. Makefileの中身がどういう意味なのかは、AIに聞いてみましょう
3. ウインドウを2つ立ち上げ、片方で、make run_server1、もう一方で、make run_client1と、コマンドをうつと、result-perf1.csvファイルができます。
4. ウインドウを2つ立ち上げ、片方で、make run_server2、もう一方で、make run_client2と、コマンドをうつと、result-perf2.csvファイルができます。
5. tcp_perf1は、本来測定したい性能測定になっていません。どういうプログラムなのかAIに聞きながら理解しましょう。
